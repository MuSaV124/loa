# LOSTARK 노드 추천 계산기 v1.0.7

## v1.0.7 변경사항
- 사이트 이름을 `LOSTARK 노드 추천 계산기`로 변경
- 검색 결과에서 API 버전 표시 제거
- API 원본 JSON 확인용 박스 제거
- 자동 추출 결과 영역 추가
- Version 1.0.6 표시

## 유지 기능
- 캐릭터 검색
- 직업/서버/아이템 레벨 표시
- 치명/특화/신속 표시
- 진화/깨달음/도약 포인트 표시


## v1.0.7 변경사항
- 검색 중 버튼을 `검색...`으로 변경하고 중복 클릭을 방지했습니다.
- 검색 결과 순서를 직업/아이템 레벨/서버 → 치명/신속/특화 → 진화/깨달음/도약으로 정리했습니다.
- 진화/깨달음/도약 노드 문구에서 진화형 피해, 적에게 주는 피해, 추가 피해, 치명타 적중률, 치명타 피해, 공격속도, 이동속도 수치를 추출합니다.
- 개발자 모드와 원본 API 박스는 추가하지 않았습니다.
