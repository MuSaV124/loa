# LOSTARK 진화 노드 추천 계산기 v1.6.0

## 변경사항

- 깨달음/도약 UI 제거
- 진화 노드 전용 구조로 단순화
- 검색된 현재 아크패시브 노드 표시
- DB에 매칭되는 진화 노드는 자동 선택
- 사용자가 진화 노드를 직접 변경하면 화면의 계산 수치가 즉시 변경
- 화면 표시 수치 추가
  - 치명타 확률
  - 치명타 피해
  - 진피
  - 추피
  - 적주피
- 추천 계산식에 추피/적주피 배율 반영
- 5티어 추천 TOP 결과 유지

## 계산 방식

기준값 입력 → 현재 선택 진화 노드 적용 → 현재 표시 수치 산출 → 5티어 후보를 하나씩 적용 → 현재 대비 증감률 표시

## 주의

- 현재 DB 계산식은 5티어 노드 중심입니다.
- 1~4티어는 API에서 검색된 노드를 보여주는 용도이며, 해당 노드 계산식이 없으면 수치에는 반영되지 않습니다.
- 뭉툭한 가시는 치명타 확률 80% 제한 및 초과 치적의 진화형 피해 전환을 반영합니다.
