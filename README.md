# lostark-node-recommend-calculator v4.7.3

## 변경 사항

- 추천 점수에 비뭉가 치적 초과 보정 추가
  - 뭉툭한 가시가 아닌 후보가 치적 100%를 넘으면 추천 점수에서만 감점
  - 초과 치적 1%당 -0.3%, 최대 -3%
  - 실제 기대값 계산은 기존처럼 치적 100% 상한 처리 유지
- 추천 조합 표시 줄임말 적용
  - 끝없는 마나 → 끝마
  - 금단의 주문 → 금주
  - 예리한 감각 → 예감
  - 한계 돌파 → 한돌
  - 최적화 훈련 → 최훈
- 4.6.8 필터 수정 유지
  - 쿨감 효과 제외 → 최적화 훈련 제외
  - 주력기 마나 사용 안함 → 끝없는 마나 / 금단의 주문 / 마나 용광로 제외
- API 원본 기대값 고정 비교 유지
- 주력기 단타 보정 유지
  - 뭉가 후보 추천 점수만 -2.5%

## 주의

추천 점수 보정은 추천 순위용입니다. 계산요약과 실제 기대값 계산식은 변경하지 않습니다.
