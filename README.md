# Lost Ark 진화 노드 추천 계산기 v2.4.0

진화 노드 전용 계산기입니다.

## v2.4.0 변경점
- 아드레날린 적용 체크박스 추가
- 체크 시 입력한 아드 치적/공증만 계산 반영
- 팔찌 치적/치피 API 자동 추출 및 입력칸 반영
- 팔찌 치적/치피 수동 보정 가능

## v2.0.0 변경점

- v1.8.0 기준 업그레이드
- 아드레날린 입력칸 추가
  - 기본 치명타 확률 +20%
  - 기본 공격력 증가 +5.4%
- 공이속 입력칸 추가
  - 기본 114%
  - 기본 100% + 만찬 5% + 서폿 진화노드 9% 기준
- 최종 계산 수치에 아드 공증/공이속 표시
- 기대값 점수에 아드 공격력 증가 반영
- 진화 노드 이미지 및 카드 UI 유지

## 이미지 위치

진화 노드 이미지는 아래 경로에 넣으면 됩니다.

```txt
public/images/evolution/
```

`public/data/evolution.json`의 `iconImage` 경로와 파일명이 맞으면 카드에 자동 표시됩니다.
