# LostArk 4티어/5티어 추천 계산기 v3

## v3 변경사항
- Vercel 504 GATEWAY_TIMEOUT 방지용 최소 조회 버전
- 조회 API를 우선 2개로 축소
  - 프로필: 직업명/스탯/아이템레벨
  - 아크패시브: 진화/깨달음/도약 및 효과 문구
- 각 LostArk API 호출에 7초 타임아웃 적용
- API가 HTML 에러를 반환해도 화면에서 원인을 보기 쉽게 처리
