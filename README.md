# LOSTARK 노드 추천 계산기 v1.3.0

## 변경사항

- 티어별 선택 UI 추가
  - 진화 1~5티어
  - 깨달음 1~4티어
  - 도약 1~2티어
- 현재 선택 노드 효과를 먼저 제거한 뒤 후보 노드를 다시 적용하는 계산 구조 추가
- 진화 5티어 노드 계산 데이터 추가
  - 뭉툭한 가시
  - 입식 타격가
  - 인파이팅
  - 음속 돌파
  - 마나 용광로
  - 안정된 관리자
- 추천 TOP 결과 표시
- 현재 대비 증감률 표시
- 드롭다운 수동 수정 및 즉시 재계산 지원

## 계산 방식

1. 사용자가 입력한 최종 스탯을 기준으로 잡습니다.
2. 현재 선택된 노드 효과를 제거합니다.
3. 제거된 기본값에 후보 노드를 하나씩 다시 적용합니다.
4. 치명타 기대값 × 진화형 피해 × 기타 최종 피해 증가를 기준으로 점수를 계산합니다.
5. 현재 선택 대비 증감률을 표시합니다.

## 주의

- 뭉툭한 가시는 치명타 확률 80% 제한 및 초과 치적의 진화형 피해 전환을 반영합니다.
- 입식 타격가는 6중첩 유지 기준으로 계산합니다.
- 음속 돌파와 마나 용광로는 조건을 만족했을 때의 최대값 기준입니다.
- 깨달음/도약은 직업별 데이터가 추가되면 같은 구조로 계산됩니다.
