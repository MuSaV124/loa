# Lost Ark Node Recommend Calculator v4.8.1-fixed

## 변경 사항
- 딜러 각인서 효과를 범용 툴팁 숫자 파싱에서 각인명/레벨 기반 고정 규칙 테이블로 변경
- 아드레날린 6중첩 기준 반영: Lv.3 공격력 +5.4%, 치명타 적중률 +20%
- 원한의 보스/레이드 몬스터 피해 증가를 계산기 적주피로 환산
- 저주받은 인형을 공격력 증가가 아니라 적주피 계열로 반영
- 어빌리티 스톤 각인 보너스 효과를 각인서 본체 효과와 분리 반영
- 어빌리티 스톤 기본 공격력 +1.5%는 스톤 툴팁의 기본 공격력 보너스로 별도 반영
- 돌격대장은 이동속도 증가량 × 전환율로 동적 계산
