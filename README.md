## v4.9.4

- 뭉툭한 가시 사용 시 최종 치적 120% 초과분에 대해 1%p당 추천값 -0.5% 페널티 적용.
- 추천 옵션에 `마나 부족 직업` 체크박스를 추가했습니다.

  - 끝마: Lv.1 +0.5% / Lv.2 +1.0%
  - 금주: Lv.1 +0.3% / Lv.2 +0.6%
  - 무마: Lv.1 +0.4% / Lv.2 +0.8%
- 이 보정은 공식 딜증이 아니라 추천 순위용 휴리스틱이며, `주력기 마나 사용 안함` 체크 시 비활성화됩니다.


## 변경 사항
- 버전 표기를 4.9.4으로 갱신.
- 각인서 피해 분류 수정.
  - `적에게 주는 피해`, `보스 및 레이드 몬스터에게 주는 피해`만 적주피로 합산.
  - 슈퍼 차지, 속전속결, 바리케이드, 안정된 상태, 타격의 대가, 기습의 대가, 결투의 대가, 돌격대장, 마나 효율 증가는 조건부 피해로 분리하여 적주피에 합산하지 않음.
- 슈퍼 차지 유물 4레벨 +21%가 적주피에 들어가던 문제 수정.
- 아드레날린 각인서 채택 시 적용값 표시 보강.
  - 치명타 적중률 및 6중첩 기준 공격력 증가를 각인서 적용 내역에 표시.
- 어빌리티 스톤 조건부 각인 보너스도 적주피와 분리.

## 기대 결과 예시
원한 / 아드레날린 / 슈퍼 차지 / 예리한 둔기 / 저주받은 인형 조합에서:
- 치피: 예리한 둔기 기준값만 반영
- 적주피: 원한 + 저주받은 인형만 반영
- 슈퍼 차지 피해량: 조건부 피해로 표시, 적주피에는 미합산
- 아드레날린: 치적 + 공격력 표시 및 체크박스 적용
